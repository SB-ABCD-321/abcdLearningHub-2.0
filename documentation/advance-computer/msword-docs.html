Chapter 1 · Beginner

Getting Started — The Basics

Learn to launch Word, understand the interface, and create your very first document. These are the essential skills every Word user must have.

Launching Microsoft Word

On Windows: Click the Start Menu → type "Word" → click Microsoft Word from results.

On Mac: Finder → Applications → Microsoft Word → double-click to open.

💡
Tip
Pin Word to your taskbar (Windows: right-click → Pin to Taskbar) or Dock (Mac: right-click → Keep in Dock) for one-click access.

Understanding the Interface

ElementDescription
Title BarShows the document name and the application's window controls (Minimize, Maximize, Close).
RibbonThe tabbed toolbar containing all commands organized by category: Home, Insert, Layout, References, etc.
Quick Access ToolbarA small toolbar above the Ribbon for frequently used commands: Save, Undo, Redo.
Document AreaThe large white workspace where you type and edit your content.
Status BarAt the bottom: shows page number, word count, language, and zoom controls.
RulerHorizontal and vertical guides for margins, indents, and tab stops.
Navigation PaneSidebar for jumping to headings or pages (View → Navigation Pane).

The Ribbon Tabs

  • Home — Font, paragraph alignment, styles, find & replace
  • Insert — Tables, pictures, shapes, charts, headers, footers, page numbers
  • Design — Document themes, colors, fonts, watermarks
  • Layout — Margins, orientation, columns, breaks, paragraph spacing
  • References — Table of contents, footnotes, citations, bibliography
  • Mailings — Mail merge, envelopes, labels
  • Review — Spelling, track changes, comments, compare documents
  • View — Switch views, zoom, arrange windows

Chapter 2 · Beginner

Text Formatting

Formatting transforms plain text into a professional, readable document. Learn to control how every character and paragraph appears on the page.

Character Formatting

Character formatting applies to selected text. Access it from Home → Font group or with shortcuts:

ShortcutEffect
Ctrl + BBold — makes text heavier and darker
Ctrl + IItalic — slants text to the right
Ctrl + UUnderline — adds a line beneath text
Ctrl + DOpen Font dialog for all settings
Ctrl + ] / [Increase / Decrease font size by 1pt
Ctrl + SpaceClear all character formatting
Shift + F3Toggle case: UPPER → lower → Title

Font Settings Explained

  • Font Family — The typeface (Times New Roman, Calibri, Arial). Serif fonts suit print; sans-serif suits screens.
  • Font Size — Measured in points (pt). Body text: 10–12pt; headings: 14–24pt.
  • Highlight — Colored background like a marker. Home → Highlight.
  • Strikethrough — Line through text; shows deleted content.
  • Subscript/Superscript — Below (H₂O) or above (x²) the baseline.

Paragraph Formatting

ShortcutEffect
Ctrl + ECenter align
Ctrl + LLeft align (default)
Ctrl + RRight align
Ctrl + JJustify (even left and right)
Ctrl + MIndent paragraph right
Ctrl + 1Single line spacing
Ctrl + 2Double line spacing
Ctrl + 51.5 line spacing

Working with Styles

Styles are saved formatting presets applied with one click. Always use built-in Heading styles (Heading 1, Heading 2, Heading 3) for section titles — this unlocks automatic Table of Contents generation.

✨
Best Practice
Never format headings manually. Always use Heading 1/2/3 styles. To modify: right-click the style → Modify → change settings → OK.

Format Painter

  1. 1 Format some text (bold, colored, size 14).
  2. 2 Click anywhere in that formatted text.
  3. 3 Click the Format Painter brush in Home → Clipboard.
  4. 4 Drag over any other text to apply the same formatting.
  5. 5 Double-click the brush to apply to multiple locations. Press Esc when done.
Practical Task 2
Format a Professional Business Letter
Objective: Apply character and paragraph formatting to create a polished business letter.
  1. 1 Open a new document. Type your name and address at the top.
  2. 2 Add recipient's address and a salutation below.
  3. 3 Write three short body paragraphs, then a closing.
  4. 4 Select your name → Ctrl+B → increase to 14pt.
  5. 5 Select body paragraphs → right-click → Paragraph → 1.5 line spacing, 12pt after.
  6. 6 Select all (Ctrl+A) → change font to Calibri 12pt.
  7. 7 Save as Business_Letter.docx.
Chapter 3 · Intermediate

Page Layout & Document Setup

Page layout controls how your document appears on a physical or digital page. Proper layout is critical for professional documents, academic submissions, and print materials.

Page Size & Orientation

  • Page Size: Layout → Size → choose Letter, A4, Legal, or More Paper Sizes for custom.
  • Portrait: Standard tall orientation — most documents.
  • Landscape: Wide orientation — ideal for wide tables and charts. Layout → Orientation → Landscape.
  • Mixed orientations: Use Section Breaks to apply different orientations to individual pages.

Margins

PresetDescription
Normal1 inch all sides — standard for most documents
Narrow0.5 inch all sides — maximizes content area
Wide2 inches left/right — for drafts with margin notes
MirroredFor double-sided (book-style) printing
CustomLayout → Margins → Custom Margins — set exact values

Page Breaks & Section Breaks

  • Page Break (Ctrl + Enter) — Forces content to the next page. Most common.
  • Next Page Section Break — New section on a new page; allows different headers, footers, and margins per section.
  • Continuous Section Break — New section on the same page; used to mix column layouts.
  • Column Break — Jumps to the next column in a multi-column layout.
⚠️
Important
To delete a break, click Ctrl + Shift + 8 to show formatting marks, click on the break, then press Delete.

Headers & Footers

  1. 1 Double-click at the very top of a page to enter header editing mode.
  2. 2 Type your header content. Use Tab to jump to the center or right.
  3. 3 Check Different First Page in the Header & Footer tab for a unique cover page header.
  4. 4 Double-click the main document area to exit header editing.

Page Numbers

  1. 1 Insert tab → Page Number → choose position (Top, Bottom, Margin).
  2. 2 Select a style from the gallery.
  3. 3 To start from a specific number: Insert → Page Number → Format Page Numbers → Start at: [number].
  4. 4 To remove: Insert → Page Number → Remove Page Numbers.
Practical Task 3
Set Up a Professional Report Layout
Objective: Configure margins, headers, footers, and page numbers for a multi-page report.
  1. 1 New document → type "ANNUAL PERFORMANCE REPORT" with Title style.
  2. 2 Add 5 sections with Heading 1: Executive Summary, Introduction, Data, Recommendations, Conclusion.
  3. 3 Layout → Margins → Top: 1", Bottom: 1", Left: 1.25", Right: 1".
  4. 4 Double-click top of page → Header → type "Annual Report | Confidential" → tab right → add date.
  5. 5 Insert → Page Number → Bottom Center. Check "Different First Page".
  6. 6 Save as Annual_Report_Template.docx.
Chapter 4 · Intermediate

Tables — Organizing Information

Tables are one of Word's most powerful tools for organizing structured data, comparing items, building schedules, and laying out complex information clearly.

Inserting a Table

  • Grid Selection: Insert → Table → hover over the grid to select rows × columns, then click.
  • Insert Table Dialog: Insert → Table → Insert Table → specify exact rows and columns.
  • Draw Table: Insert → Table → Draw Table → draw outline and cells with your cursor.
  • Quick Tables: Insert → Table → Quick Tables → choose a pre-built calendar or matrix.
  • Convert Text to Table: Select tab-separated text → Insert → Table → Convert Text to Table.

Table Navigation & Editing

ActionHow To
TabMove to the next cell (creates new row at end)
Shift + TabMove to the previous cell
Alt + Shift + ↑/↓Move entire row up or down
Right-click → InsertAdd rows or columns above, below, left, or right
Layout → Merge CellsCombine selected cells into one
Layout → Split CellsDivide one cell into multiple rows/columns
Layout → AutoFitAuto-adjust column widths to content or window

Formatting Tables

  • Table Styles: Click a design in the Table Design gallery for instant professional styling.
  • Header Row: Tick "Header Row" in Table Design for special first-row formatting.
  • Banded Rows: Alternating shading for easy reading.
  • Borders: Customize style, weight, and color via the Borders dropdown.
  • Cell Alignment: Layout tab → Alignment group (top-left to bottom-right).
  • Table Properties: Right-click → Table Properties for precise column widths, row heights, and text wrap.
Practical Task 4

  1. 1 Insert a 5-column, 8-row table.
  2. 2 Headers: Task, Owner, Start Date, End Date, Status.
  3. 3 Fill 7 rows with project tasks (Planning, Design, Dev, Testing, etc.).
  4. 4 Select header row → Shading → dark blue → change text to white, bold, centered.
  5. 5 Layout → AutoFit → AutoFit Contents.
  6. 6 Add Status values: Completed / In Progress / Pending.
  7. 7 Save as Project_Timeline.docx.
Chapter 5 · Intermediate

Images, Shapes & Visual Elements

Visual elements dramatically enhance your documents. Word lets you insert and manipulate images, shapes, icons, SmartArt diagrams, and charts.

Inserting Images

  • From File: Insert → Pictures → This Device → navigate and select.
  • Online Pictures: Insert → Pictures → Online Pictures → search Bing.
  • Screenshot: Insert → Screenshot → select a window or Screen Clipping.
  • Stock Images: Insert → Pictures → Stock Images (Microsoft 365 only).
  • Paste: Copy any image from any source and paste directly with Ctrl + V.

Image Formatting

  • Wrap Text: Controls text flow around the image. Options: In Line with Text, Square, Tight, Through, Top and Bottom, Behind Text, In Front of Text.
  • Crop: Picture Format → Crop. Drag handles to trim edges.
  • Resize: Drag corner handles (maintains aspect ratio) or enter exact dimensions.
  • Corrections: Adjust brightness, contrast, and sharpness.
  • Artistic Effects: Apply pencil sketch, watercolor, mosaic, and more.
  • Alt Text: Right-click → Edit Alt Text for accessibility descriptions.
💡
Tip
Hold Shift while dragging a corner handle to resize an image while maintaining its original aspect ratio — preventing distortion.

SmartArt Diagrams

SmartArt converts bullet lists into professional diagrams. Insert → SmartArt → choose a category:

CategoryBest For
ListNon-sequential or grouped items
ProcessSequential steps or workflows
CycleRecurring or circular processes
HierarchyOrg charts, reporting structures
RelationshipConnections between concepts
MatrixQuadrant or grid relationships
PyramidProportional or hierarchical data

Design an Illustrated Report Section

  4. 4 Add 5 steps: Planning, Design, Development, Testing, Launch.

Chapter 6 · Intermediate

References — TOC, Footnotes & Citations

The References tab provides essential tools for academic and long-form professional documents: automatic navigation aids, footnotes, and source citation management.

Table of Contents

⚠️
Prerequisite
For a TOC to work, you MUST use Word's built-in Heading styles (Heading 1, Heading 2, etc.) for all your section titles. Manual bold text will not be recognized.
  1. 1 Apply Heading 1/2/3 styles to all section titles.
  2. 2 Click at the beginning of the document where you want the TOC.
  3. 3 References tab → Table of Contents → choose Automatic Table 1 or 2.
  4. 4 The TOC appears with clickable links and page numbers.
  5. 5 To update: click the TOC → Update Table → Update entire table.

Footnotes & Endnotes

ShortcutAction
Ctrl + Alt + FInsert a Footnote (appears at page bottom)
Ctrl + Alt + DInsert an Endnote (appears at document end)

To convert: right-click a footnote → Convert to Endnote, or use the Footnote & Endnote dialog to convert all at once.

Citations & Bibliography

Word supports APA, MLA, Chicago, IEEE, Harvard, and more.

  1. 1 References → Manage Sources → New → choose Source Type (Book, Article, Website, etc.).
  2. 2 Fill in Author, Title, Year, Publisher → OK.
  3. 3 To cite: place cursor in text → References → Insert Citation → select source.
  4. 4 To insert Bibliography: References → Bibliography → choose format.
  5. 5 The bibliography updates automatically as you add more sources.
Practical Task 6
Build an Academic Paper Structure
Objective: Create an academic paper with TOC, footnote citations, and APA bibliography.
  1. 1 New doc. Page 1: Title style → "The Impact of Technology on Modern Education".
  2. 2 Page 2 (page break): Insert Automatic Table of Contents.
  3. 3 Pages 3+: 5 sections (Heading 1): Abstract, Introduction, Literature Review, Methodology, Conclusion.
  4. 4 Add 2 subsections (Heading 2) within any section.
  5. 5 Add a footnote in Introduction: Ctrl + Alt + F.
  6. 6 Add 2 sources via Manage Sources. Set style to APA. Insert in-text citations.
  7. 7 Insert Bibliography at the end. Update Table of Contents. Save as Academic_Paper.docx.
Chapter 7 · Intermediate

Review & Collaboration Tools

Word's collaboration features make it essential for teams. Track every edit, leave contextual comments, and protect documents from unauthorized changes.

Spelling & Grammar

  • Real-time checking: Red wavy underline = spelling; blue/gold = grammar. Right-click for suggestions.
  • Full check: F7 or Review → Spelling & Grammar.
  • Thesaurus: Right-click a word → Synonyms, or Shift + F7.
  • Word Count: Review → Word Count (or see count in the status bar).
  • AutoCorrect: File → Options → Proofing → AutoCorrect Options to manage auto-fixes.

Track Changes

Track Changes records every edit — additions, deletions, formatting changes — attributed to a specific author with a timestamp.

ActionHow To
Ctrl + Shift + EToggle Track Changes on/off
Review → Next / PrevNavigate between tracked changes
Review → AcceptAccept the current tracked change
Review → RejectReject and undo the current tracked change
Accept AllAccept every change in the document at once
Display for ReviewToggle: All Markup / Simple Markup / No Markup / Original

Comments

  • Add comment: Select text → Review → New Comment (Ctrl + Alt + M) → type.
  • Reply: Click comment → Reply icon → type response.
  • Resolve: Click comment → Resolve. Collapses but remains viewable.
  • Delete: Right-click → Delete Comment, or Review → Delete.
  • @mentions: In Microsoft 365, type @ in a comment to tag a colleague directly.

Document Protection

  • Restrict Editing: Review → Protect → Restrict Editing. Allow no changes, tracked changes only, comments only, or form-filling only.
  • Password Protect: File → Info → Protect Document → Encrypt with Password.
  • Mark as Final: File → Info → Protect Document → Mark as Final. Signals completion (not fully locked).
Practical Task 7
Collaborative Review Simulation
Objective: Practice Track Changes and Comments as a document reviewer.
  1. 1 Open Business_Letter.docx from Task 2.
  2. 2 Enable Track Changes: Ctrl + Shift + E.
  3. 3 Delete the first sentence — notice the strikethrough red text.
  4. 4 Add a new sentence at the end of paragraph 2 — appears as underlined blue text.

Chapter 8 · Advanced

Advanced Features — Power User Tools

These advanced tools transform Word from a text editor into a document automation platform. Master them to multiply your productivity.

Mail Merge

Mail Merge creates personalized copies of a document (letters, emails, labels) by combining a template with a data source like an Excel spreadsheet.

  1. 1 Write your template letter with placeholder spaces.
  2. 2 Mailings → Start Mail Merge → Letters.
  3. 3 Select Recipients → Use an Existing List → select your Excel file.
  4. 4 Mailings → Insert Merge Field → click field names (e.g., FirstName, City).
  5. 5 Preview Results to review merged letters.
  6. 6 Finish & Merge → Edit Individual Documents → All.

Templates

  • Using templates: File → New → search for Resume, Invoice, Report, etc.
  • Creating templates: Design and format a document → File → Save As → Word Template (*.dotx).
  • Normal template: Normal.dotm is the base for every blank document. Modify it to change Word's default font, margins, and styles permanently.

Macros

🔐
Security Note
Only enable macros from trusted sources. Malicious macros can harm your system. File → Options → Trust Center → Macro Settings to configure.
  1. 1 View → Macros → Record Macro. Give it a name (no spaces).
  2. 2 Optionally assign a keyboard shortcut. Click OK — recording begins.
  3. 3 Perform the actions you want to automate.
  4. 4 View → Macros → Stop Recording.
  5. 5 Run it anytime: View → Macros → View Macros → select → Run.
  6. 6 To edit code: View → Macros → Edit (opens Visual Basic Editor).

Advanced Find & Replace

  • Basic: Ctrl + H to find and replace text across the document.
  • Match case: Only replaces text with exact capitalization.
  • Whole words only: Won't match "cat" inside "category".
  • Wildcards: Enable to use ? (any char), * (any string), [a-z] (range).
  • Replace formatting: Click Format at the bottom to replace with specific font or style.
  • Special characters: Click Special for paragraph marks (^p), tabs (^t), and more.
Practical Task 8
Mail Merge Newsletter
Objective: Create a personalized newsletter using Mail Merge with an Excel subscriber list.
  1. 1 Create an Excel file with columns: FirstName, LastName, City, MembershipLevel. Add 5+ rows.
  2. 2 New Word doc → type "The Monthly Digest" as header.
  3. 3 Type salutation: "Dear ," (leave space for merge field).
  4. 4 Mailings → Start Mail Merge → Letters → Select Recipients → your Excel file.
  5. 5 Position cursor after "Dear" → Insert Merge Field → FirstName, then LastName.
  6. 6 Insert City and MembershipLevel fields in the body paragraph.
  7. 7 Preview Results → cycle through recipients. Finish & Merge → Edit Individual Documents.
Chapter 9 · Beginner

Saving, Exporting & File Management

Understanding how to properly save and export your documents ensures they are accessible, shareable, and protected.

File Formats

FormatUse Case
.docxDefault Word format. Compatible with Word 2007+. Use for all editable documents.
.docLegacy format for Word 97–2003. Use only for very old system compatibility.
.pdfFile → Save As → PDF. Universal, non-editable. Best for sharing final documents.
.dotxWord Template. Save reusable document structures.
.txtPlain text only — no formatting. For raw text transfer.
.rtfRich Text Format — retains basic formatting; compatible with many apps.
.odtOpenDocument — for LibreOffice/OpenOffice compatibility.

AutoSave & AutoRecover

  • AutoSave: In Microsoft 365, documents on OneDrive/SharePoint save every few seconds automatically.
  • AutoRecover: Saves a recovery copy every 10 minutes by default. Configure: File → Options → Save → AutoRecover every [x] minutes.
  • Recover Unsaved: File → Info → Manage Document → Recover Unsaved Documents.

Document Metadata & Privacy

Documents contain hidden metadata: author name, revision history, comments. Before sharing externally, clean this data:

  1. 1 File → Info → Check for Issues → Inspect Document.
  2. 2 Select categories to check (Comments, Personal Info, Revisions, etc.).
  3. 3 Click Inspect → Remove All next to categories you want to clean.
Chapter 10 · Advanced

Pro Tips & Best Practices

Expert-level strategies to work faster, produce higher-quality documents, and avoid the most common mistakes that slow professionals down.

Productivity Tips

  • Navigation Pane: View → Navigation Pane. Drag headings to reorganize entire sections instantly.
  • Tell Me box: Press Alt + Q and type what you want to do — Word finds the command.
  • Split View: View → Split. See two parts of the same document simultaneously.
  • Focus Mode: View → Focus (Microsoft 365) — hides everything but the document.
  • Repeat last action: F4 repeats the most recent action on any selection.
  • Customize Ribbon: File → Options → Customize Ribbon. Build your own tab.
  • Multiple windows: View → New Window. Open same document in two windows side by side.

Formatting Best Practices

  • Always use Styles instead of manual formatting for consistency.
  • Use Section Breaks (not blank lines) to separate major document sections.
  • Set spacing in the style definition rather than pressing Enter multiple times.
  • Use Tab stops instead of multiple spaces for precise alignment.
  • Limit font families to two per document — one for headings, one for body.
  • Never use the spacebar for indentation — use Tab or paragraph indent settings.
  • Compress images before sharing: Picture Format → Compress Pictures.

Troubleshooting

ProblemSolution
TOC not updatingEnsure headings use built-in Heading styles. Right-click TOC → Update Field → Update entire table.
Bad paste formattingUse Ctrl + Shift + V → Keep Text Only to strip external formatting.
Images moving unexpectedlySet Wrap Text to "In Line with Text" or use absolute anchoring.
Spell check not workingSelect all → Review → Language → Set Proofing Language → uncheck "Do not check spelling".
Document is very slowCompress all images: Picture Format → Compress Pictures → select All pictures.
Track Changes won't turn offFile → Options → Trust Center → Trust Center Settings → Privacy Options.

Advanced Project A
Create a Complete Professional Resume
Skills: Tables, Styles, Columns, Formatting, Page Layout, PDF Export · Est. Time: 45–60 minutes

  5. 5 Under EXPERIENCE: 2-column borderless table — company/dates left, job title/bullets right.
  6. 6 Under SKILLS: Layout → Columns → 3 for a compact skills grid.
  7. 7 Ensure it fits on one page. Run F7 spell check. Print Preview review.

  2. 2 Page 2 (new section): Automatic Table of Contents.
  3. 3 Pages 3–10: 7 sections with Heading 1 titles, 150+ words each.
