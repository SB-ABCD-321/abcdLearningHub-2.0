Gmail

Google's powerful email platform used by over 1.8 billion people. Manage email, organize with labels, set up filters, and boost productivity.

What is Gmail?

Gmail is Google's free email service that provides 15 GB of free storage (shared with Drive and Photos), powerful search, smart filters, labels, and integration with all Google Workspace apps. It works across web, Android, and iOS.

ℹ️
Gmail uses Labels instead of traditional folders. A single email can have multiple labels, unlike a folder where it can only exist in one place.

Core Features

📥 Inbox Categories

Gmail auto-sorts emails into Primary, Social, Promotions, Updates, and Forums tabs using machine learning.

🏷️ Labels & Filters

Create custom labels and auto-filters to organize incoming mail automatically based on sender, subject, or content.

⭐ Stars & Importance

Mark emails as important or starred. Gmail also learns your email patterns and auto-marks important messages.

🔍 Advanced Search

Use powerful search operators to find any email instantly — by date, sender, attachment type, label, and more.

Writing & Composing Emails

1
Open Compose Window

Click the Compose button (bottom-left) or press C on your keyboard to open a new email draft.

2
Add Recipients

Type in the To field. Use CC (carbon copy) for informational recipients, BCC for hidden recipients.

3
Write Subject & Body

A clear subject line increases open rates. Use the formatting toolbar to bold, italicize, or add lists to your email body.

4
Add Attachments or Drive Files

Click the paperclip 📎 to attach files, or the Drive icon to share Google Drive files as links (recommended for large files).

5
Schedule or Send

Click Send or use the dropdown arrow next to Send to choose Schedule send for a specific date/time.

Setting Up Filters

Filters automatically process incoming emails. Go to Settings → See all settings → Filters and Blocked Addresses → Create a new filter.

💡
Pro Tip: Create a filter for newsletters by searching list:unsubscribe then applying label "Newsletters" and skipping the inbox. This keeps your inbox clean without unsubscribing.

Google Drive

Cloud storage and file synchronization service. Store, share, and access files from any device. 15 GB free storage shared across Google services.

Understanding Google Drive

Google Drive is your personal cloud file system. It stores not just uploaded files but also native Google files (Docs, Sheets, Slides) which don't count against storage. You can access Drive via browser, desktop app, or mobile app.

🗂️ My Drive

Your personal storage area. Organize files in folders, sub-folders. You own everything here.

👥 Shared Drives

Team-based drives where files belong to the organization, not individuals. Great for collaboration.

🔗 Shared with Me

Files others have shared with you. These don't consume your storage quota.

⭐ Starred

Quick access to your most important files across all drives.

Google Docs

A full-featured online word processor. Create, edit, and collaborate on documents in real time. Compatible with Microsoft Word (.docx).

Document Creation & Formatting

Google Docs auto-saves every change to Google Drive. Start a new document at docs.new or from your Drive. Every formatting option is in the toolbar or Format menu.

Styles & Headings

Use heading styles (H1, H2, H3) consistently — they auto-generate your Document Outline (View → Show outline), making navigation easy in long documents. Styles can be customized via Format → Paragraph styles → Options → Save as my default styles.

Real-time Collaboration

Multiple people can edit simultaneously. Each collaborator gets a unique colored cursor. Use Comments for suggestions and questions without altering the document itself.

1
Add a Comment

Select text → Right-click → Comment, or press CtrlAltM. Tag a collaborator with @email to notify them.

2
Suggest Edits (Track Changes)

Switch to Suggesting mode (top-right dropdown: Editing → Suggesting). Your changes appear as suggestions others can accept or reject.

3
View Version History

Go to File → Version history → See version history to see all changes ever made, who made them, and restore any version.

Tables & Structure

Insert tables via Insert → Table. Right-click a table for options to add/delete rows, merge cells, or distribute columns evenly. Tables are great for organizing comparative information.

Page Setup & Document Settings

Access via File → Page setup. Set paper size (A4 vs Letter), orientation, margins, and page color. For pageless documents, choose Format → Switch to pageless — great for online reading.

ℹ️
Google Docs now supports Markdown input. Type **bold** or # Heading and it auto-converts. Enable under Tools → Preferences → Automatically detect Markdown.

Google Sheets

A powerful online spreadsheet application. Analyze data, create charts, automate with formulas, and collaborate in real time.

Spreadsheet Basics

A Google Sheet consists of Cells (referenced by column letter + row number, e.g., A1), organized in Rows and Columns across one or more Sheets (tabs). Start a new sheet at sheets.new.

ℹ️
Cell references: A1 = relative (changes when copied), $A$1 = absolute (stays fixed), A$1 = mixed (row fixed, column relative).

Essential Formulas & Functions

Mathematical & Statistical

Math Functions 
=SUM(A1:A10)              # Add up a range
=AVERAGE(B2:B20)         # Calculate mean
=MIN(C1:C100)            # Find minimum value
=MAX(C1:C100)            # Find maximum value
=COUNT(A:A)              # Count numbers in column
=COUNTA(A:A)             # Count non-empty cells
=COUNTIF(A:A, "Yes")     # Count cells matching criteria
=SUMIF(A:A, "Q1", B:B)  # Sum B where A = "Q1"
=ROUND(3.14159, 2)       # Round to 2 decimal places → 3.14
=ABS(-42)                # Absolute value → 42

Lookup Functions

Lookup Functions 
# VLOOKUP: Find data in a table by row
=VLOOKUP("John", A:D, 3, FALSE)
# Finds "John" in column A, returns value from column 3 (C)

# XLOOKUP: Modern replacement for VLOOKUP (more flexible)
=XLOOKUP("John", A:A, C:C, "Not found")

# INDEX + MATCH: Powerful combination
=INDEX(C:C, MATCH("John", A:A, 0))

# HLOOKUP: Find data in a table by column (horizontal)
=HLOOKUP("Revenue", A1:E5, 3, FALSE)

Text Functions

Text Functions 
=CONCATENATE(A1, " ", B1)    # Join text: "First Last"
=A1&" "&B1                    # Shorter way to concatenate
=LEFT(A1, 3)                  # First 3 characters
=RIGHT(A1, 5)                 # Last 5 characters
=MID(A1, 2, 4)                # Characters 2-5
=LEN(A1)                      # Character count
=UPPER(A1)                    # UPPERCASE
=LOWER(A1)                    # lowercase
=PROPER(A1)                   # Title Case
=TRIM(A1)                     # Remove extra spaces
=SUBSTITUTE(A1, "old", "new") # Replace text
=TEXT(A1, "MM/DD/YYYY")       # Format dates as text

Logical Functions

Logical Functions 
=IF(A1>100, "High", "Low")         # If-then-else
=IF(A1="", "Empty", A1)           # Check if empty
=IFS(A1>90,"A", A1>80,"B", TRUE,"C") # Multiple conditions
=AND(A1>10, B1<100)             # Both conditions must be true
=OR(A1="Yes", B1="Yes")         # Either condition can be true
=NOT(A1="Done")                 # Reverse a condition
=IFERROR(A1/B1, 0)             # Return 0 if formula errors
=ISBLANK(A1)                   # Check if cell is empty

Pivot Tables

Pivot tables summarize large datasets. Select your data → Insert → Pivot table. In the editor, drag fields to Rows, Columns, Values, and Filters. For example, drag "Salesperson" to Rows and "Revenue" to Values to see revenue per salesperson instantly.

Charts & Data Visualization

Select data → Insert → Chart. Google Sheets suggests chart types. You can use Line, Bar, Column, Pie, Scatter, Area, Histogram, and more. Customize in the Chart editor under "Customize" tab.

Conditional Formatting

Highlight cells automatically based on values: select cells → Format → Conditional formatting. Use single-color rules (e.g., red if < 0) or color scales (gradient from low to high).

💡
Data Validation: Restrict cell input to a dropdown list via Data → Data validation. This prevents data entry errors in shared sheets.

Google Slides

Cloud-based presentation software. Create beautiful, interactive presentations and present from any device.

Creating Presentations

Start at slides.new. Choose a theme from the panel, or go to Slide → Edit theme to customize fonts, colors, and backgrounds for every slide at once. Themes ensure consistent styling across all slides.

Slide Layouts & Masters

Go to Slide → Edit master to edit the Master slide template and individual layouts. Changes here propagate to all slides using that layout — a massive time-saver for consistent branding.

Working with Elements

📝 Text Boxes

Insert → Text box. Drag to resize. Double-click to edit. Use the format toolbar to style text with fonts, sizes, and colors.

🖼️ Images

Insert → Image → Upload, URL, Camera, Google Drive, Google Photos, or web search directly within Slides.

📐 Shapes & Lines

Insert → Shape for hundreds of shapes. Hold Shift while drawing to constrain proportions. Use arrows and connectors for diagrams.

📊 Charts & Tables

Insert → Chart to embed live Sheets charts that update when data changes. Insert → Table for structured data.

Transitions & Animations

Transitions affect how slides enter. Go to Slide → Transition. Animations control individual elements within a slide. Select an element → Insert → Animation. Set entrance, emphasis, or exit animations with timing control.

⚠️
Use animations sparingly. Over-animating makes presentations feel unprofessional and distracts from content. One simple transition (like Fade) applied to all slides is usually sufficient.

Presenting Like a Pro

1
Use Presenter View

Click the dropdown arrow next to "Present" → Presenter view. You'll see your current slide, next slide preview, speaker notes, and a timer on your screen, while the audience only sees the presentation.

2
Add Speaker Notes

Click "Click to add speaker notes" at the bottom of each slide. These are visible only to you in Presenter View and can be exported when printing.

3
Q&A Mode

During presentation, click Present → Audience tools → Start new Q&A. Share a link with your audience so they can submit questions anonymously.

Google Meet

High-quality video conferencing. Host meetings for up to 100 participants (free) or 500+ (Workspace). No software installation required.

Starting & Joining Meetings

1
Start Instantly

Go to meet.google.com and click "New meeting" → "Start an instant meeting". Share the meeting link or code with participants.

2
Schedule in Calendar

Create a Google Calendar event and click "Add Google Meet video conferencing". The Meet link auto-populates in the invite.

3
Join with a Code

Click "Use a meeting code" on meet.google.com or enter the 10-character code (e.g., abc-defg-hij) to join any meeting.

Meet Features

🖥️ Screen Sharing

Click "Present now" → choose to share your entire screen, a specific window, or a single browser tab (best for audio/video).

✋ Reactions & Polls

Use emoji reactions, raise hand, and polls (Activities panel) to engage participants without interrupting.

🎨 Virtual Backgrounds

Click "..." → Visual effects to blur your background, choose a preset background, or upload a custom image.

📝 Meeting Notes

Click Activities → Meeting notes to automatically create a linked Google Doc for the meeting — it appears in the Calendar event.

⏺️ Recording

Click Activities → Recording (Workspace required). Recordings save to Google Drive in the organizer's folder automatically.

🔴 Live Streaming

Google Calendar

Smart scheduling and time management. Create events, schedule meetings, set reminders, and share calendars with your team.

Calendar Views

Switch between Day, Week, Month, Year, Schedule (agenda view), and 4-day views from the top-right menu. The Schedule view is particularly useful for getting an overview of upcoming events in list format.

Creating & Managing Events

1
Create an Event

Click any time slot on the calendar, or press C. Fill in title, date/time, location, and description. Click "More options" for advanced settings.

2
Invite People

Add guests by email. You'll see their availability if they're in your organization. Use the View attendees' calendars feature to find a time everyone is free.

3
Add Reminders & Notifications

Set email or notification reminders. Default is 30 minutes before — you can add multiple reminders (e.g., 1 day before + 30 minutes before).

4
Set Recurring Events

In "Does not repeat" dropdown, choose daily, weekly, monthly, annually, or set a custom recurrence like "every 2 weeks on Monday and Wednesday".

Multiple Calendars

Create separate calendars for different areas: Work, Personal, Birthdays, Travel. Color-code each. Share work calendars with colleagues so they can see your availability. Toggle visibility with the checkbox next to each calendar.

Google Chat

Real-time messaging platform for teams. Direct messages, group conversations, and Spaces for ongoing project communication.

Direct Messages vs. Spaces

💬 Direct Messages

One-on-one or small group chats. Best for quick questions, informal conversations, or private discussions. Messages are not threaded.

🏠 Spaces

Persistent, topic-based channels. Invite team members, share files, and use threaded conversations to keep discussions organized by topic.

Key Features

In Spaces, conversations are threaded — reply to specific messages to keep topics separate. Use @mentions to notify specific people. Share Google Drive files directly in chat — they appear as rich previews. Use /meet command to instantly start a Meet call with the Space participants.

💡
React to messages with emoji reactions by hovering over a message and clicking the 😊 icon. This is great for quick acknowledgments without sending a "Thanks!" message.

Admin Console

Central management for Google Workspace administrators. Manage users, security, apps, devices, and billing.

Key Admin Tasks

👤 User Management

Add, remove, suspend users. Reset passwords. Manage 2-step verification. Set up user aliases and groups.

🔒 Security Center

Monitor security alerts, review OAuth app access, enable advanced protection, and manage data export/retention.

📱 Device Management

Enroll and monitor company devices. Apply security policies, wipe lost devices, and manage mobile device access.

📊 Reports & Audit

View usage reports for all apps, audit logs for user activity, and email delivery reports.

Organizational Units (OUs)

OUs let you apply different settings to different groups. For example, your Sales OU might have YouTube access enabled while your standard OU does not. Nest OUs for granular control: Company → Sales → Enterprise Sales.

Google Drive Tips

• Press . (period) on any file in Drive to get a quick preview without opening it.
• Shift+Z to add a shortcut to a file in multiple folders without duplicating it.
• Use Priority page in Drive — it learns what files you use most and surfaces them automatically.
• Right-click a file → Make a copy to duplicate Google files without losing the original's sharing settings.

Gmail Power Features

• Undo Send: Enable in Settings with a 30-second delay — your last line of defense against accidental sends.
• Snooze emails: Right-click an email → Snooze to have it reappear at a specific time. Perfect for "I'll deal with this later."
• Templates: Enable in Settings → Advanced → Templates. Save frequently sent email templates for one-click reuse.
• Confidential Mode: When composing, click the lock icon to send self-expiring emails that can't be forwarded or copied.

Sheets Advanced

• ArrayFormula: Apply a formula to an entire column without copying it: =ARRAYFORMULA(A2:A*B2:B)
• QUERY function: Use SQL-like queries on your data: =QUERY(A:D,"SELECT A,C WHERE B='Sales' ORDER BY C DESC")
• IMPORTRANGE: Pull data from another Sheet: =IMPORTRANGE("spreadsheet_url","Sheet1!A:D")
• Google Finance: Get live stock data: =GOOGLEFINANCE("GOOG","price")

Docs Hidden Features

• @-mention to insert: In Docs, type @ to insert people, files, dates, events, and more — turns Docs into a connected hub.
• Explore panel: CtrlAltShiftI opens an AI-powered research panel without leaving your doc.
• Word count shortcut: CtrlShiftC shows live word count.
• Bookmarks: Insert → Bookmark, then Insert → Link → select the bookmark to link between sections of a long document.

🤖
Gemini AI in Workspace: Google Workspace now integrates Gemini AI across Docs, Sheets, Slides, Gmail, and Meet. Look for the ✨ Gemini icon to: draft emails, summarize long documents, create Sheets formulas from plain English, generate slide content, and take meeting notes automatically.
